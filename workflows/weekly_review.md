# Weekly Review Workflow

Run at the end of each week (Friday afternoon) or start of the next (Monday morning).

## Steps

### 1. Reflect on the Past Week — All Three Roles

Ask Arnold:

**Solopreneur:**
- "What were the solopreneur wins this week? What didn't get done?"
- Review project progress against `context/projects.md`

**Bally's:**
- "How was the Bally's week? Anything notable — team wins, challenges, decisions?"
- Update `context/ballys.md` if Arnold shares priorities or changes

**Church:**
- "How was Sunday? How did the sermon go?" (if Arnold preached)
- "Any ministry leadership updates?"
- Update `context/lifecity_church.md` if relevant

Log wins and incomplete items in `memory/logs/` under the current week's entry.

### 2. Review Active Projects (Solopreneur)

- Read `context/projects.md`
- For each active project, ask: "Any status updates?"
- Update project statuses in `context/projects.md`

### 3. Process Decisions

- Ask: "Any decisions made this week worth capturing? (Any role)"
- Append to `memory/decisions.md`

### 4. Review Goals

- Read `context/goals.md`
- Check: Are current projects aligned to current goals across all roles?
- Flag any drift: "Your solopreneur projects are heavy this week — are Bally's and church getting enough attention?"

### 5. Plan Next Week

- Run `workflows/weekly_planning.md` which includes:
  - **Critical question:** "Are you preaching next Sunday?"
  - Solopreneur priorities for next week
  - Known Bally's commitments
  - Sermon prep schedule (if preaching)
  - Day-specific defaults (Monday: ReguBrief, Wednesday: Blog, Friday: ChurchPromptDirectory)

### 6. Content Planning

- Use `skills/blog_writing/SKILL.md` to ideate the Wednesday blog post based on the past week's wins, struggles, or decisions
- Note the proposed blog topic in `memory/logs/` for reference

### 7. Inbox Zero

- Run `skills/task_capture/SKILL.md` on any remaining `inbox/` items
- Confirm inbox is clear before closing

## Completion

Report: "Weekly review complete. Here's the summary: [wins across all 3 roles, decisions, next week plan, preaching status]"
