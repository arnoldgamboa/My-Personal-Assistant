# Resources

This folder contains templates and reference documents used across the PA system.

## Structure

```
resources/
├── templates/          ← Reusable document templates
│   ├── meeting_notes.md
│   ├── project_brief.md
│   └── weekly_log.md
└── references/         ← Static reference material
    └── (add as needed)
```

## Templates

### Meeting Notes (`templates/meeting_notes.md`)
Captures structured notes during and after meetings.

### Project Brief (`templates/project_brief.md`)
One-page summary of a project for stakeholders.

### Weekly Log (`templates/weekly_log.md`)
Standard template for weekly log entries in `memory/logs/`.
