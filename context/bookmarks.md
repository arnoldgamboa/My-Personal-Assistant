# Bookmarks

> Links worth saving for later reference, research, or discussion.
> This is separate from Todoist. Todoist stays for daily actionable work only.

---

## How to File Bookmarks

- Use one primary category per bookmark
- Add 2-5 short tags for searchability
- Keep notes focused on why the link matters

### Entry Format

- **[Title]**
  Category: [Primary category]
  Type: [Website / Video / Article / Repo / Tool / Reference]
  Tags: [tag1, tag2, tag3]
  Notes: [Why this is useful]
  URL: [Link]
  Added: YYYY-MM-DD

---

## Categories

- `AI Tools` — models, APIs, agents, AI products

- `Product & App Ideas` — inspiration for products, features, workflows
- `Research & Search` — search engines, research workflows, discovery tools
- `References & Learning` — tutorials, explainers, documentation, reusable learning material
